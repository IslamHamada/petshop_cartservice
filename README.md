# Cart Service

Handles cart management for the PetShop platform.

## Description
This service manages cart items and relies on the Product Service to validate product availability and details.

## Tech Stack
- Java 17  
- Spring Boot 3.x  
- Spring Data JPA  
- MySQL  

Requires:
- Product Service running
